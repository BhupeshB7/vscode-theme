# bhupeshb7-coder-dark-theme

bhupeshb7-coder-dark-theme is a sleek and modern dark theme for Visual Studio Code. It is designed to reduce eye strain and improve code readability with a carefully chosen color palette.

![Theme Preview](./bg.png)

## Installation

1. Open the Extensions sidebar in VS Code.
2. Search for `bhupeshb7-coder-dark-theme`.
3. Click **Install**.
4. Click **Reload** to activate the theme.

Alternatively, you can manually install the theme:

1. Download the latest release of the theme from the [GitHub repository](https://github.com/your-repo).
2. Unzip the downloaded file and copy the contents to the `.vscode/extensions` directory.

## Features

- Dark background with high-contrast colors
- Easy on the eyes, suitable for long coding sessions
- Consistent color scheme for better focus
